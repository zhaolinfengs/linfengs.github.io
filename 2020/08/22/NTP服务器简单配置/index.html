NTP服务器简单配置

系统环境：rhel 7.6

简单思路：

1.服务器地址：10.10.128.1
  客户端地址：10.10.128.2-3
2.准备工作：
（1）关闭防火墙
（2）关闭selinux
（3）搭建yum仓库
3.配置服务器
4.配置客户端
5.测试

一、设置网络

    set server:    ifconfig ens33 10.10.128.1/24 up
    set client: ifconfig ens33 10.10.128.2/24 up

二、准备工作

    1.设置防火墙
        systemctl stop firewalld.service      //关闭防火墙
        systemctl status firewalld.service     //查看防火墙状态

    2.关闭selinux
        setenforce 0

    3.搭建Yum仓库
        vim /etc/yum.repo.d/a.repo
        内容：
        [a]
        name=a
        gpgcheck=0
        enabled=1
        baseurl=file:///mnt/cdrom

三、配置服务器

    1.安装NTP组件
        yum install -y ntp 
    2.修改主配置文件
