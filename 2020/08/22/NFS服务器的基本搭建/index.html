NFS服务器的基本搭建

环境:

redhat 7.6 server
redhat 7.6 client
vmware workstation

一、搭建yum仓库

1.挂载本地光盘
 (1)创建挂载点
 mkdir /mnt/cdrom

 (2)编写挂载表
  vim /etc/fstab
 内容：
  /dev/sr0     /mnt/cdrom        iso9660        defaults    0 0
 (3)刷新挂载表
  mount -a

 (4)查看是否挂载成功
  lsblk

2.创建仓库文件
  vim /etc/yum.repo.d/a.repo    //"a.repo"为仓库文件，需要手动创建
 内容：
    [a]
    name=a
    gpgcheck=0
    enabled=1
    baseurl=file:///mnt/cdrom
保存退出wq

2.查看仓库文件是否生效
    yum repolist

二、安装nfs组件

yum install -y nfs*

三、编写主配置文件

vim /etc/exports
内容：
/tmp/redhat     192.168.8.0/24(rw,sync)        //主配置文件分为两列，第一列为共享的资源（目录），第二列为可以接受共享的用户后面括号是（共享选项）
保存退出wq

四、重启服务

systemctl restart nfs

五、测试

切换到client系统下：

showmount -e 192.168.8.200            //查看目标路径的共享资源
mount -t nfs     192.168.8.200:/tmp/redhat /mnt/nfs    //将目标用户共享的资源/tmp/redhat挂载到本地的 /mnt/nfs下
